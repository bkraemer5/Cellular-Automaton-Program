INTRO:
	The purpose of this project is to create the cella ant #12
	algorithm using Javascript on an HTML webpage for display.
	Cella ant #12 function is determined by the number 1100 in
	binary--where 1 means turn left, and 0 means turn right.
	For visuals, each bit is assigned a color: 0 = black, 1 =
	red, 2 = yellow, 3 = blue. The algorithm is displayed using
	these colors to easily see each move that is being made.

	}

FEATURES:
	- Colors change as the mover moves across the board
	- Mover rotates left/right depending on color on board
	- Can easily customize the amount of moves made by the mover
